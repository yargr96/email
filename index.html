Из инвентаря потребуется следующее:


1) Духовой шкаф с регулировкой температуры для выпечки бисквитов.

2) Миксер
Ручной с 2 венчиками (для небольших объемов),
планетарный (для больших объемов, если готовите на заказ).
При выборе обращать внимание на мощность (ручной берем хотя бы 400-500 Вт)

3) Блендер погружной
С чашей-мельничкой – для приготовления фасолевой муки

4) Металлическое раздвижной кольцо для выпечки. Формы для бисквитов.

5) Антипригарный тефлоновый коврик или силиконизированный пергамент. Бумажный пергамент.

6) Мешки и насадки кондитерские
Мешки одноразовые. Из полиэтилена или силиконизированные. Размер зависит от количества теста/начинки.
Насадки – металлические. Только они дают четкую форму:

Французкая трубочка 12-14 мм диам. Отверстия.

Ровная, круглая 13-15 мм

7) Термометр
Электронный игольчатый – для работы с жидкостями (варка сиропов, фруктовых начинок и пр.)

8) Венчики, силиконовые лопатки

9) Шпатель для работы с размером около 20 см.
Из кондитерского или строительного магазина (берите качественный, нержавеющий)

10) Подложки для тортов /пирожных.
Размер чуть больше диаметра формы.
С их помощью удобно транспортировать торты.

11) Сотейник или кастрюля с толстым дном.

Небходимые ингредиенты для первого занятия:


Миндальная мука 110г
Сахарная пудра 225г
Яичный белок 125
Сахар 50г
Пищевой краситель
Абрикосы (можно консервированные) 200г
Лимонный сок 1/4
Ванильный стручок
